This repository contains the complete documentation for the robotics project.

- The structure is organized by component and includes the following main sections:

- CAD Model: Contains the computer-aided designs and models of the robot.

- Electrical: Includes diagrams, schematics, and documentation related to the electrical components and wiring.

- Mathematical Models: Contains the theoretical development and equations describing the robot's kinematics and dynamics.

- Simulation: Covers the files and results from the model testing and simulations.
